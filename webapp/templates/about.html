🧠 About This Project

🍽️ What is this?

This is a fun, beginner-friendly Convolutional Neural Network (CNN) that can recognize 4 types of food from images: Pizza 🍕, Pasta 🍝, Salad 🥗, and Donuts 🍩!

🔧 Tech Stack

  • Machine Learning: TensorFlow 2.x (Keras), NumPy
  • Image Processing: PIL, OpenCV
  • Web Interface: Flask, HTML, CSS, JavaScript
  • Visualization: Matplotlib

🎯 Features

  • 🧠 Custom CNN architecture with 3 convolutional layers
  • 📊 Confidence meter with fun messages
  • 👨‍🍳 Random chef tips for each food type
  • 🔗 Recipe links for cooking inspiration
  • 👃 Easter egg "smell detector" (coming soon!)
  • 📱 Clean, responsive web interface

🚀 How it Works

  1. Data Preprocessing: Images are resized to 128x128 pixels and normalized
  2. CNN Architecture: 3 conv layers + pooling + batch normalization + dropout
  3. Training: Model learns from sample food images with data augmentation
  4. Prediction: New images are classified with confidence scores

📈 Model Performance

The model uses early stopping and learning rate reduction to achieve optimal performance. Training typically reaches 80-90% accuracy on validation data with the sample dataset.
